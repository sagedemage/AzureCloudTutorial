Introduction:

Vi is a visal text editor included by default on Unix operating systems.
It is a character-based screen editor without a GUI. One of Vi's major draws
is its ubiquity and accessibility. You don't need a desktop enviornment to use it,
and it is typically already installed on your system.
Vi gives users a robust editing experience without ever having to leave the keyboard's
home row. With a dedicated command mode, Vi utilizes the full keyboard for tasks
like navigating a file, manipulating text, searching, and replacing phrases

How to open Vi editor

You can open up Vi with the vi command. Generally, if your system uses vim, this command still links
to it, but you can also use the vim command in that case. You could alternatively, open Vi with a file
using the vi [example-file.txt], replace example-file with the path to the desired file

Vi modes

Vi has two distinct modes, getting familiar with them helps you get the most out of Vi

  • Normal mode: This mode is also known as "command mode" and is the mode where you enter
    most of your Vi commands. Key pressed in normal mode are evaluated as commands and not as text
    entered on the file you are editing.
  • Insert mode: This mode is your main place for entering text. Here, your key presses tend to
    operate how you would expect in a text editor. You can enter this mode by pressing the i key
    while you are in normal mode. Alternatively, you can press the o key to insert a new line
    and immediately enter Insert mode.

Common Vi commands in normal mode

  • :e ~/example-file (opens a file in the current user's home directory)
  • :w ~/example-file (writes changes to a file, if you want to specify the file to save to,
    add the path after the command

Navigation

You can navigate text in Vi using the arrow keys, whether in normal or insert mode. However, in
normal mode, you can also use the keys in the list below. Some of these keys help keep you from
laving the home row for the arrow keys, but most add handy and efficient navigation shortcuts

  • j (moves the cursor down a line)
  • k (moves the cursor up a line)

Undo and Redo

To undo a change, use the u command. You can then redo a change with :redo

Copy, Cut, and Paste

The base Vi command for copying is y, the list shows you a variety of ways you can quickly define
what text gets copied

Search and Replace

Vi uses the / command to start a search. Once you enter the / command, you can begin entering your
search text. Note that Vi's searches are case sensitive. Pressing enter once you have entered your search
text takes you to the first instance of that text in the file. You can then navigate the search results using the
n command to go forward and the N command to go backward

How to exit Vi

To exit Vi, press esc if you're in normal mode then use the :q command.
Additionally, if you want to save your changes and exit in a single command,
you can combine the two commands using, :wq
You can also exit Vi without saving your changes. To do so, add a ! to the end of the quit command :q!
